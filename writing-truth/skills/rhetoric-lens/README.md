# rhetoric-lens

处理文字的修辞层——检测、具象、造类比。方法源自李笑来《写作的真相》第五课「修辞正义」。第一原则:**思想 > 文采**,修辞只为更好地传递感知,不为显摆。

**Status:** v0.1.0 — project-scope, manual install.

## What it does

围绕四种重要修辞(具象、排比、类比、隐喻)提供三种模式:

1. **检测(读别人的)** — 给一篇文章或一本书,扫出全部具象/排比/类比/隐喻,逐条摘原文并拆结构(类比标出 X 本体 / A 喻体 / 相似点 / 跨界 / 边界),汇成"修辞收集本"。
2. **具象(改自己的空话)** — 把抽象/概括的句子改"具体具体再具体",两条路径(概括→具体、抽象→具体),并附治 writing block 的天马行空小游戏。
3. **类比/隐喻(为概念造桥)** — 用公式 `X≈A → 讲A → X 不言自明` 跨界生成类比,带边界检查(A 是否更好懂 / 是否只在某层面相似 / 防滥用),可简化为隐喻。

## Invoke

```
/writing-truth:rhetoric-lens <粘贴文本或书名章节 / 一句空话 / 一个待解释的概念>
```

先认清是"检测(读别人的)"还是"创作(写自己的)"。

## Pairs with

- `logic-template-lens`(第四课·逻辑轴)—— 它看并列/递进/转折骨架,本技能看修辞层,可叠用。

## Constraints

守住"思想 > 文采":若其实是内容/逻辑有问题,先提醒,不用修辞掩盖。检测模式引用原文时,逐字措辞以原书为准。
